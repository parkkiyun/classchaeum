## 🖥️ 앱 실행 방법

### 방법 1: 자동 실행 스크립트 (권장)
```bash
# Windows에서 배치 파일 실행
start-app.bat
```

### 방법 2: 수동 실행
```bash
# 의존성 설치 (최초 1회만)
npm install

# 개발 서버 시작
npm run dev
```

### 방법 3: Electron 데스크탑 앱
```bash
# 브라우저와 별도로 Electron 앱 실행
npm run electron:dev
```
